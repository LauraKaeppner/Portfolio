Magic Materials

This project deals with the conceptualization, design and implementation of procedural materials that can change dynamically during the runtime of a game in a game engine. Due to the different controllable states of the individual materials, they can be used in the engine as the foundation for a puzzle game mechanic. Using the "Magic Materials", a synergy of design and technology is created which is relevant for the development of dynamic game environments.

As a result, a playable prototype for the possible use of the materials in the game engine as well as loop videos to illustrate their transformation and a digital artbook to document the design process were created.
